# 📘 Assignment: Testing and Debugging in Python

## 🎯 Objective

Practice writing simple tests and debugging Python code by verifying function behavior, using assertions, and fixing a broken starter program.

## 📝 Tasks

### 🛠️ Write Test Functions

#### Description
Create test functions to verify the behavior of two Python helpers.

#### Requirements
Completed program should:

- Implement `calculate_average(numbers)` to return the average of a list of numbers.
- Implement `is_prime(n)` to return `True` for prime integers and `False` otherwise.
- Create `test_calculate_average()` and `test_is_prime()` functions that use `assert` statements to verify expected results.
- Include at least three assertions for each test function.

### 🛠️ Debug the Starter Program

#### Description
Review and correct the provided starter program so it runs correctly and all tests pass.

#### Requirements
Completed program should:

- Fix syntax and logic errors in `starter-code.py`.
- Use assertions or print debug checks to confirm the program behavior.
- Run the corrected starter code without errors and show the final test output.
